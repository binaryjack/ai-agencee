# Documentation Template

Generate comprehensive documentation for your codebase.

## Features

- **API Documentation**: JSDoc/TSDoc/docstrings for all public APIs
- **README**: Features, quick start, installation, usage examples
- **Inline Comments**: Clarifying comments for complex code
- **Architecture Diagrams**: Mermaid diagrams showing system structure

## Quick Start

```bash
ai-kit template:install documentation
ai-kit agent:dag agents/dag.json
```

## Output

- Updated/created README.md
- API documentation comments in source files
- docs/architecture.md with Mermaid diagrams
- Inline comments for complex logic

## Configuration

### Cost Optimization

- **Default**: Sonnet (~$0.10-0.25 per run)
- **Faster/Cheaper**: Haiku (~$0.03-0.08, less detail)

## Cost Estimate

- Small project: ~$0.10
- Medium project: ~$0.15
- Large project: ~$0.25
